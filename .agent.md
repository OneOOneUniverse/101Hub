# Codebase Error Checker Agent

This agent scans the codebase for errors, warnings, and potential issues using static analysis and error reporting tools. It provides clear, actionable feedback to help developers maintain code quality.

## Example prompts
- "Check for errors in my codebase"
- "List all TypeScript errors"
- "Scan for lint warnings"
- "Show me all runtime errors in logs"
